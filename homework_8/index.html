BACGROUND

Это универсальное CSS свойство позволяет одновременно задать несколько параметров фона, а именно определить цвет, установить фоновое изображение, задать начальное положение фонового изображения, определить как будет повторяться фоновое изображение, установить будет ли прокручиваться фоновое изображение вместе с содержимым и масштабировать фоновое изображение согласно заданным размерам. Так же значением свойства background может быть указана градиентная заливка фона.

  • BACKGRAUND-COLOR
  • BACKGRAUND-IMAGE
  • BACKGRAUND-REPEAT
  • BACKGRAUND-POSITION
  • BACGRAUND-SIZE
  • BACGRAUND-ATTACHMENT
  • MULTIPLE BACGROUND

BACKGRAUND-COLOR

Определяет цвет фона элемента

По умолчанию (прозрачный)

CSS код:
.block{
background: transparent;
}
Или:
.block{
background-color: transparent;
}

Цветной

CSS код:
.block{
background: #18b5a4;
}
Или:
.block{
background-color: #18b5a4;
}

Цветной (rgb)

CSS код:
.block{
background: rgb(24,181,164);
}
Или:
.block{
background-color:rgb(24,181,164);
}

Цветной (rgba)

CSS код:
.block{
background:rgb(204,109,21,0.6);
}
Или:
.block{
background-color:rgb(204,109,21,0.6);
}

BACKGRAUND-IMAGE

Устанавливает фоновое изображение для элемента либо задает градиентную заливку

УСТАНАВЛИВАЕТ КАРТИНКУ

CSS код:
.block{
background:rgb(25, 127, 182) url('../img/1-tesla.jpg');
}
Или:
.block{
background-image: url('../img/1-tesla.jpg');
}

УСТАНАВЛИВАЕТ ЛИНЕЙНЫЙ ГРАДИЕНТ

Градиентом называют плавный переход от одного цвета к другому, причём самих цветов и переходов между ними может быть несколько. С помощью градиентов создаются самые причудливые эффекты веб-дизайна, например, псевдотрёхмерность, блики, фон и др. Также с градиентом элементы смотрятся более симпатично, чем однотонные.
CSS код:
.block{
background-image: linear-gradient(to left, rgba(30,87,153,0), rgba(30,87,153,0.8), rgba(30,87,153,1), rgba(30,87,153,1), rgba(41,137,216,1), rgba(30,87,153,1), rgba(30,87,153,1), rgba(30,87,153,0.8),rgba(30,87,153,0));
}

УСТАНАВЛИВАЕТ РАДИАЛЬНЫЙ ГРАДИЕНТ

Радиальные по своему принципу похожи на линейные градиенты, но один цвет переходит в другой не вдоль прямой линии, а словно круги по воде вокруг точки.
Для составления градиентов и получения готового кода используется GRADIENT GENERATOR CSS код:
.block{
background:background: radial-gradient(ellipse at center, rgba(184,225,252,1) 0%,rgba(169,210,243,1) 10%,rgba(144,186,228,1) 25%,rgba(144,188,234,1) 37%,rgba(144,191,240,1) 50%,rgba(107,168,229,1) 58%,rgba(162,218,245,1) 83%,rgba(189,243,253,1) 100%);
}

BACGROUND-REPEAT

Определяет, как будет повторяться фоновое изображение, установленное с помощью свойства background-image
(по умолчанию repeat).

no-repeat - Устанавливает одно фоновое изображение в элементе без его повторений, положение которого определяется свойством background-position
(по умолчанию в левом верхнем углу).
background:greenyellow url('../img/car-1jpg.jpg') 0 0 no-repeat;
repeat - Фоновое изображение повторяется по горизонтали и вертикали.
background:greenyellow url('../img/car-1jpg.jpg') 0 0 repeat;
repeat-x - Фоновый рисунок повторяется только по горизонтали.
background:greenyellow url('../img/car-1jpg.jpg') 0 0 repeat-x;
repeat-y - Фоновый рисунок повторяется только по вертикали.
background:greenyellow url('../img/car-1jpg.jpg') 0 0 repeat-y;
space - Изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство.
background:greenyellow url('../img/car-1jpg.jpg') 0 0 space;
round - Изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются.
background:greenyellow url('../img/car-1jpg.jpg') 0 0 space;
repeat и паттерное изображение Для уменьшения веса страницы если возможно для заполнения фона используется паттерное изображение (это маленькая картинка вырезаная таким образом что её начало есть продолжением её конца)
background:greenyellow url('../img/bg_repeat.jpg') 0 0 repeat;
Паттерное изображение
repeat-x, repeat-y и паттерное изображение
background:url('../img/bg_repeat.jpg') 0 0 repeat-x,greenyellow url('../img/bg_repeat.jpg') 0 0 repeat-y;
Паттерное изображение

BACKGRAUND-POSITION

Задает начальное положение фонового изображения, установленного с помощью свойства background-image, где первое значение (проценты-лутше при адаптиве, px, слово[left, center, right, top, bottom]) определяет расположение по горизонтали, второе - вертикали.

По умолчанию(top left, 0 0)
CSS код:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 0 0 no-repeat;
}
Или:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') top left no-repeat;
}
Или:
.block{
background-color:rgb(153, 142, 142);
background-image: url('../img/car-2png.png');
background-repeat: no-repeat;
background-position: 0 0;
}
Слева по центру блока(left, 0 50%)
CSS код:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') left no-repeat;
}
Или:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 0 50% no-repeat;
}
Или:
.block{
background-color:rgb(153, 142, 142);
background-image: url('../img/car-2png.png');
background-repeat: no-repeat;
background-position: 0 50%;
}
По центру блока(center, 50% 50%, 50%)
CSS код:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') center no-repeat;
}
Или:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 50% no-repeat;
}
Или:
.block{
background-color:rgb(153, 142, 142);
background-image: url('../img/car-2png.png');
background-repeat: no-repeat;
background-position: 50% 50%;
}
В правом нижнем углу блока(right bottom, 100% 100%)
CSS код:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') right bottom no-repeat;
}
Или:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 100% 100% no-repeat;
}
Или:
.block{
background-color:rgb(153, 142, 142);
background-image: url('../img/car-2png.png');
background-repeat: no-repeat;
background-position: 100% 100%;
}
Размещенеие по заданной позиции(50% 100%)
CSS код:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 50% 100% no-repeat;
}
Или:
.block{
background:rgb(153, 142, 142) url('../img/car-2png.png') 50% 100% no-repeat;
}
Или:
.block{
background-color:rgb(153, 142, 142);
background-image: url('../img/car-2png.png');
background-repeat: no-repeat;
background-position: 50% 100%;
}

BACGRAUND-SIZE

Масштабирует фоновое изображение согласно заданным размерам. (значение) - Задает размер в любых доступных для CSS единицах. (проценты) - Задает размер фоновой картинки в процентах от ширины или высоты элемента.Либо специальным значением (auto, contain, cover).

Pазмер картинки по умолчанию (auto auto)
Если задано одновременно для ширины и высоты (auto auto), размеры фона остаются исходными;
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 50% /auto auto no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 50%;
background-repeat: no-repeat;
background-size: auto auto;
}
Размер картинки по ширине блока (100% auto)
Если задан размер только для ширины картинки (100px auto), то размер высоты вычисляется автоматически исходя из пропорций картинки.
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 100% /100% auto no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 100%;
background-repeat: no-repeat;
background-size: 100% auto;
}
Размер картинки по высоте блока (auto 100% )
Если задан размер только для высоты картинки (auto 100px ), то размер ширины вычисляется автоматически исходя из пропорций картинки.
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') -150 100% /auto 100% no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: -150px 100%;
background-repeat: no-repeat;
background-size: auto 100% ;
}
Размер (100% 100% )
Расстягивает картинку на всю ширину и высоту блока
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 0 /100% 100% no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 0;
background-repeat: no-repeat;
background-size: auto 100% ;
}
Параметр contain
Масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока. То есть по меньшей стороне картинки
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 0 /contain no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 0;
background-repeat: no-repeat;
background-size: contain ;
}
Параметр cover
Масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока. То есть по большей стороне картинки
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 0 /cover no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 0;
background-repeat: no-repeat;
background-size: cover ;
}
Конкретный размер (300px 400px )
Установленные конкретные размеры из двух значений через пробел задает ширину и высоту фоновой картинки
CSS код:
.block{
background:rgb(201, 27, 14,0.5) url('../img/car-1jpg.jpg') 0 0 /300px 400px no-repeat;
}
Или:
.block{
background-image: url('../img/car-1jpg.jpg');
background-position: 0 0;
background-repeat: no-repeat;
background-size: 300px 400px ;
}

BACGRAUND-ATTACHMENT

Устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента.

По умолчанию (scroll)
Позволяет перемещаться фону вместе с окном браузера background: CSS код:
.block{
background: url('../img/car-3jpg.jpg') 0 0 no-repeat scroll;
}
Или:
.block{
background-image: url('../img/car-3jpg.jpg');
background-repeat: no-repeat;
background-position: 0 0;
background-attachment: scroll;
}
fixed
Делает фоновое изображение элемента неподвижным, а так же устанавливает фоновое изображение относительно окна браузера.
CSS код:
.block{
background: url('../img/car-3jpg.jpg') 0 0 no-repeat fixed;
}
Или:
.block{
background-image: url('../img/car-3jpg.jpg');
background-repeat: no-repeat;
background-position: 0 0;
background-attachment: fixed;
}

MULTIPLE BACGROUND

Несколько картинок в одном блоке
background: url('../img/car1-1.jpg') 50% no-repeat,
url('../img/bg_repeat.jpg') 0 0 repeat-x,
url('../img/bg_repeat.jpg') 0 100% repeat-x,
url('../img/bg_repeat.jpg') 0 0 repeat-y,
url('../img/bg_repeat.jpg') 100% 0 repeat-y ;
